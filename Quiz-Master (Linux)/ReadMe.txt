WELCOME TO THE QUIZ MASTER!

Save this complete folder in any directory on your system.

Navigate your self to the directory through terminal using the command: cd filepath
where 'filepath' is the path for the directory form the Home Directory.

Open the terminal

Execute the Following Command on the command line:

make quiz (Hit Enter)
./quiz (On next promt and then Hit Enter)

You can Create new Categories and add as many questions as you want.
Also compete with your friends in V/S Mode!

Enjoy The Quiz!!
