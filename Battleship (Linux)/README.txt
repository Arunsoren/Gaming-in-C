WELCOME TO THE BATTLESHIP!

Save this complete folder in any directory on your system.

Navigate your self to the directory through terminal using the command: cd filepath
where 'filepath' is the path for the directory form the Home Directory.

Execute the following command: make battleship
And then : ./battleship

Enjoy The Game!
